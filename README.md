# 🪐 Seraloc_Dev — Premium Script Hub



---

## 📌 Описание проекта

**Seraloc_Dev Premium Script Hub** — это многофункциональная, оптимизированная графическая панель управления (GUI) для тестирования механик и автоматизации в среде Roblox. Скрипт разработан на языке **Luau** с использованием продвинутых методов оптимизации (Zero Lag), встроенной плавной анимации интерфейса и кастомных настроек клавиш активации.

Интерфейс выполнен в минималистичном стиле **"Hacker Dark"** с использованием глубоких темных тонов и неоновых magenta/зеленых акцентов.

---

## ⚡ Основной функционал (Опции)

1. **Invisible Mode (Режим инвидимости):** Полностью скрывает визуальное отображение персонажа (все части тела и декали становятся прозрачными) на стороне клиента для проверки коллизий.
2. **NoClip Engine (Прохождение сквозь стены):** Отключает физическую коллизию (`CanCollide = false`) для всех частей тела персонажа через оптимизированный цикл `RunService.Stepped`.
3. **ESP Box 2D (Классический ESP):** Отображает четкие прямоугольные рамки вокруг всех игроков на сервере (Team Check отключен). Включает в себя:
   - Динамическую зеленую полосу здоровья (Health Bar) с левой стороны рамки.
   - Текстовый индикатор расстояния с автоматической конвертацией в метры (`m`) или километры (`km`).
4. **AimBot Pro (Автонаведение):** Высокоточный скрипт захвата цели в голову (`Head`) на базе `RenderStepped` и `DeltaTime`, исключающий задержки. Работает в двух режимах: по зажатию ПКМ или полностью автоматически (настраивается в коде через `RequireRightClick`).
5. **Fly Engine (Режим полета):** Плавный полет с использованием линейной физики векторов (`BodyVelocity`/ `BodyGyro`), не ломающий анимации персонажа.

## ⌨️ Управление и Горячие клавиши

* **Клавиша `[B]`** — Скрыть / Показать панель управления (с эффектом плавного изменения размера и прозрачности).
* **Клавиша `[Delete]`** — **Полная деинсталляция скрипта**. Мгновенно отключает все активные циклы, очищает кэш рисунков Drawing API с экрана, восстанавливает прозрачность персонажа и выгружает GUI из оперативной памяти (Anti-Memory Leak).
* **Кастомные бинды (Custom Keybinds):** Рядом с каждой функцией в меню находится текстовое поле (`TextBox`). Вы можете кликнуть туда и ввести любую букву (например: `G`, `F`, `X`), чтобы мгновенно привязать эту клавишу для включения/выключения данной функции.

## 🛠️ Технические особенности (Для разработчиков)

- **Режим Строгой Типизации:** Код полностью совместим со стандартом `--!strict` языка Luau, что повышает скорость компиляции и обработки математических вычислений CFrame.
- **Безопасное сворачивание (Minimize):** Кнопка `[ — ]` в шапке панели сжимает интерфейс до компактной строки заголовка, автоматически скрывая список кнопок и подвал, чтобы они не загораживали обзор во время игры.
- **Draggable UI:** Панель можно свободно перемещать по экрану, удерживая левую кнопку мыши на верхней панели (Header) с названием проекта.
